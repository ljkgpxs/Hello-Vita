##Hello_Vita

Sample program for PSVita
  
##Compiling
requisites:
- psp2sdk
- [libvita2d](https://github.com/xerpi/libvita2d)
  
##Usage
Up : Set CPU Clock + 1  
Down : Set CPU Clock - 1  
L : Circle's raid - 10  
R : Circle's raid + 10  
Start : Quit program  
Front touch two points : Change Circle's radius and position
